# Aula 01 - Instalação do ComfyUI

Nesta aula apresentamos as três formas de usar o ComfyUI — via cloud, via repositório GitHub e via instalador oficial — e realizamos a instalação local no Windows usando o instalador disponível em [https://www.comfy.org/](https://www.comfy.org/). Ao final da aula, o ComfyUI estará instalado e rodando na sua máquina, com a interface acessível pelo navegador.

## 🛠️ Requisitos e Instalação

### Requisitos de Hardware
- **GPU NVIDIA** (recomendado): placa com pelo menos 4 GB de VRAM para geração de imagens em resolução padrão.
- **CPU** (alternativa): possível, mas com desempenho significativamente inferior.
- **RAM**: mínimo 16 GB recomendado.
- **Armazenamento**: reservar ao menos 20 GB livres para o ComfyUI e os modelos.

### Instalação via Instalador Oficial (Windows)

1. Acesse [https://www.comfy.org/](https://www.comfy.org/)
2. Baixe o instalador para Windows (arquivo `.exe`).
3. Execute o instalador e siga as etapas, escolhendo o diretório de destino.
4. Ao final da instalação, abra o ComfyUI pelo atalho criado no Desktop ou pelo menu Iniciar.
5. Aguarde o carregamento dos logs no terminal.
6. Acesse a interface no navegador em: `http://127.0.0.1:8188`

### Alternativas de Instalação (não abordadas em detalhes no curso)
- **Cloud**: plataformas como RunPod ou Google Colab — sem instalação local, mas com custo por uso.
- **GitHub (manual)**: clone do repositório `https://github.com/comfyanonymous/ComfyUI` com Python e pip — para usuários avançados.

## 📂 Arquivos da Aula

Esta aula é introdutória e focada na instalação. Não há arquivos de código ou workflows para download nesta etapa.

## 🎯 O que aprendemos:

- As três formas de usar o ComfyUI: cloud, instalador e GitHub.
- Como baixar e executar o instalador oficial do site `comfy.org`.
- Como verificar se o ComfyUI está rodando corretamente acessando `http://127.0.0.1:8188` no navegador.

## 🚀 Desafio Prático

Instale o ComfyUI na sua máquina seguindo os passos demonstrados na aula. Após a instalação, abra a interface no navegador, tire um print da tela com o ComfyUI carregado e compartilhe nos comentários do vídeo. Se encontrar algum erro durante a instalação, descreva o problema nos comentários para que possamos ajudar.
